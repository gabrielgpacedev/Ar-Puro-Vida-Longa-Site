As regras

Como jogar

A meta é deixar a qualidade do ar em 100% em cada cenário. Você consegue isso fazendo duas coisas: tirando o que polui e colocando o que ajuda.

PASSO 01 👀

Leia a cena

Cada fase mostra uma rua com várias fontes de poluição: carros, ônibus velho, fumaça e lixo.

PASSO 02 🫳

Remova a poluição

Arraste cada fonte de poluição para fora da cena para limpar o ar daquele ponto.

PASSO 03 🌳

Aplique soluções

Solte as soluções na rua — árvores, transporte limpo, energia renovável — onde fazem mais efeito.

PASSO 04 💙

Devolva o céu azul

A cada acerto o ar melhora. Chegue a 100% e a rua poluída vira a rua limpa. Ar puro, vida longa!

Para o desenvolvedor — conectar o build da Unity
Quando o build WebGL estiver publicado, é só plugar a URL e o jogo aparece aqui.
  1. Na Unity: File ▸ Build Settings ▸ WebGL ▸ Build.
  2. Publique a pasta gerada num host estático — itch.io é o caminho mais fácil para jogos (já cuida da compressão); GitHub Pages, Netlify ou Vercel também servem.
  3. Copie a URL pública do jogo (o endereço do index.html do build).
  4. Abra este arquivo e cole a URL na linha const GAME_URL = "", no início do script lá embaixo.
Enquanto GAME_URL estiver vazio, esta área mostra o aviso de "em desenvolvimento". Assim que houver uma URL, o botão de play carrega o jogo num iframe. WebGL precisa ser servido por http(s) — abrir como arquivo local não funciona.
